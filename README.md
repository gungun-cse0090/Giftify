# 🎁 Giftify – E-commerce Gift Website

Giftify is a modern and user-friendly e-commerce website where users can explore and purchase gifts like cakes 🎂, chocolates 🍫, and bouquets 💐.

---

## 🚀 Features

✨ Attractive Home Page with categories  
🎂 Cakes Collection  
🍫 Chocolates Collection  
💐 Bouquet Collection  
🛒 Add to Cart functionality (with LocalStorage)  
💳 Payment Page (shows total items & price)  
🔍 Smart Search Bar (redirect-based search)  
📱 Fully Responsive Design (mobile friendly)  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- LocalStorage (for cart management)

## 💡 How It Works

- Users browse different categories (Cakes, Chocolates, Bouquets)
- Products can be added to cart
- Cart data is stored using LocalStorage
- Cart is synced across all pages
- Payment page displays total items and total price
